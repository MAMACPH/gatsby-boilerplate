<h1 align="center">
  MAMA's Gatsby default starter (based on Gatsby's starter default)
</h1>

##Development

    ```sh
    gatsby develop
    ```

    The site is now running at `http://localhost:8000`! And GraphQL can be tested at `http://localhost:8000/___graphql`_.

##Production
		
		```sh
    gatsby build
    ```
